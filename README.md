# Student Management System

A console-based C++ project to manage student records, designed for learning Object-Oriented Programming (OOP) and data structures.

## Features
- Add, Delete, Search, Display student records
- Uses STL vector for data storage

## How to Run
1. Open the project in any C++ IDE (Code::Blocks / Dev C++)
2. Compile and run main.cpp
3. Follow console instructions

## Technologies Used
- C++ programming language
- Object-Oriented Programming (OOP)
- STL vector
